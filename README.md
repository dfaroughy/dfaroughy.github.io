# faroughy.com

Personal website of Darius A. Faroughy — Research Scientist, ML / AI for Science.

Static single-page site served via GitHub Pages at https://www.faroughy.com.

## Editing

The entire site is `index.html` (HTML + CSS + JS, no build step). Edit it and push to `main`; GitHub Pages redeploys automatically within a minute or two.

## Hosting / domain

- Hosted on GitHub Pages from this repo.
- Custom domain set in **Settings → Pages**; the `CNAME` file in the repo root holds `www.faroughy.com` and must not be deleted.
- HTTPS is provisioned automatically by GitHub.
